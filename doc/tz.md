# ТЗ - Интернет-магазин

## 1. Общее описание

Интернет-магазин — это веб-приложение, позволяющее пользователям просматривать товары, добавлять их в корзину и оформлять заказы. Также предусмотрена админ-панель для управления товарами и заказами.

## 2. Основные функции

### 2.1. Для пользователей:
- Регистрация и авторизация
- Просмотр товаров (с фильтрацией и сортировкой)
- Добавление товаров в корзину
- Оформление заказа
- Просмотр истории заказов

### 2.2. Для администраторов:
- Управление товарами (добавление, редактирование, удаление)
- Управление заказами (изменение статуса)
- Управление пользователями

## 3. Требования к системе

- Фронтенд: HTML/CSS/JavaScript (React)
- Бэкенд: Java (Spring Boot)
- База данных: PostgreSQL
- REST API
- JWT-аутентификация
- CI/CD через Jenkins
- Деплой через Docker

## 4. Пользовательские роли

| Роль           | Описание                               |
|----------------|----------------------------------------|
| Покупатель     | Может просматривать товары, оформлять заказы |
| Администратор  | Может управлять товарами и заказами     |

## 5. Технические требования

- RESTful API
- Поддержка масштабирования

## 6. Системные ограничения

- Максимальное количество товаров в корзине — 100
- Время жизни сессии — 24 часа
- Поддержка 1000 одновременных пользователей

## 7. Планируемый срок выполнения

- 1-2 недели на реализацию основной функциональности
- 1 неделя на тестирование и документацию

## 8. Критерии приемки

- Все пользовательские сценарии реализованы
- API работает корректно
- Тесты проходят успешно
- Документация готова
